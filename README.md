# personality-test-app

# local development

## Service

```bash
$ cd ./service 
$ npm install
$ npm start
```
## Webapp

```bash
$ cd ./webapp 
$ npm install
$ npm start
```

* Run the webapp on [http://localhost:3000](http://localhost:3000)
* Swagger on [http://localhost:9229/api](http://localhost:9229/api)
